Core Components

SOPAC, Locum, and Insurge comprise the core elements of the Social Catalog Application Suite. Some people are confused because SOPAC is the front-end that everyone sees. In order to get it working, however, you need to also download, install and configure the two supporting software libraries, Locum and Insurge.

Locum Connectors

Locum is the engine behind SOPAC that allows it to integrate so seamlessly with your ILS. In addition to installing Locum, you will need to either download, install and configure one of these connectors or write one yourself. The connector is like a "driver" that allows Locum to connect to your ILS, which in turn allows SOPAC to do all the nifty things it does.
